Politique de Confidentialité - Arkalia CIA

Dernière mise à jour : 17 novembre 2025

1. INTRODUCTION

Arkalia CIA ("nous", "notre", "nos") s'engage à protéger votre vie privée. Cette Politique de Confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile.

2. COLLECTE DE DONNÉES

Arkalia CIA est conçue avec la confidentialité comme principe fondamental :

  • AUCUNE DONNÉE COLLECTÉE : Nous ne collectons, ne transmettons, ni ne stockons aucune donnée personnelle sur des serveurs externes.
  • STOCKAGE LOCAL UNIQUEMENT : Toutes vos données (documents, rappels, contacts) sont stockées exclusivement sur votre appareil.
  • AUCUNE ANALYSE : Nous n'utilisons pas d'outils d'analyse ou de services de suivi.
  • AUCUNE SYNCHRONISATION CLOUD : Vos données ne quittent jamais votre appareil sauf si vous choisissez explicitement de les exporter.

3. STOCKAGE DES DONNÉES

Toutes les données sont stockées localement sur votre appareil en utilisant :

  • Base de données SQLite chiffrée (chiffrement AES-256)
  • Stockage de fichiers sécurisé avec chiffrement au niveau de l'appareil
  • Aucune sauvegarde cloud ou synchronisation

4. PERMISSIONS

Arkalia CIA demande les permissions suivantes :

  • Accès au calendrier : Pour créer et gérer les rappels
  • Accès aux contacts : Pour gérer les contacts d'urgence (ICE)
  • Accès au stockage : Pour sauvegarder et gérer les documents
  • Notifications : Pour délivrer les notifications de rappels
  • Appels téléphoniques : Pour appeler les contacts d'urgence
  • Authentification biométrique : Pour sécuriser l'accès à l'application

Toutes les permissions sont optionnelles et l'application peut fonctionner avec des fonctionnalités limitées si les permissions sont refusées.

5. SÉCURITÉ DES DONNÉES

  • Chiffrement AES-256 pour toutes les données stockées
  • Sécurité au niveau de l'appareil (Keychain/Keystore)
  • Aucune transmission réseau de données personnelles
  • Traitement des données uniquement local

6. SERVICES TIERS

Arkalia CIA n'intègre pas de services tiers d'analyse, de publicité ou de collecte de données.

7. CONFIDENTIALITÉ DES ENFANTS

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations auprès d'enfants.

8. VOS DROITS

Étant donné que toutes les données sont stockées localement :

  • Vous avez un contrôle total sur vos données
  • Vous pouvez supprimer toutes les données en désinstallant l'application
  • Vous pouvez exporter vos données à tout moment
  • Aucun compte requis, aucun partage de données

9. MODIFICATIONS DE CETTE POLITIQUE

Nous pouvons mettre à jour cette Politique de Confidentialité. Les modifications seront publiées dans l'application et sur notre dépôt GitHub.
